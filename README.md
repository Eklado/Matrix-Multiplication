# Matrix multiplication using Laravel

## API:

URL: "/api/matrix/multiplication"

Method: POST

Parameters:

- "matrix1" Array of arrays of integers (Mandatory)
- "matrix2" Array of arrays of integers (Mandatory)

Security: For authorized users only
